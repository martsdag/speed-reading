# 📚 Speed Reading PDF Tool

Простое и эффективное веб-приложение для скорочтения PDF-документов. Использует метод **RSVP** (Rapid Serial Visual Presentation), который позволяет читать быстрее за счет фиксации взгляда на одном месте.

---

## ✨ Основные функции

* **Загрузка PDF:** Чтение текстового слоя напрямую из локальных файлов.
* **Умное разбиение:** Алгоритм делит каждое слово на три части, выделяя центральный символ для лучшей фокусировки.
* **Контроль скорости:** Настройка темпа чтения от 100 до 500 слов в минуту (WPM).
* **Память прогресса:** Автоматическое сохранение последней прочитанной страницы для каждого файла в `LocalStorage`.
* **Интернационализация (i18n):** Поддержка интерфейса на английском, русском и испанском языках.
* **Навигация:** Возможность переключения между страницами и остановки/возобновления чтения.

---

## 🛠 Технологический стек

* **Frontend:** Чистый JavaScript (ES6+), HTML5, CSS3.
* **Библиотека PDF:** [PDF.js](https://mozilla.github.io/pdf.js/) (v2.16.105) для парсинга документов.
* **Хранилище:** `Web Storage API` (LocalStorage) для сохранения состояния.
* **Шрифты:** Google Fonts (Poppins, Prosto One).

---
## ⚠️ Ограничения

- Приложение работает только с PDF, имеющими текстовый слой. Отсканированные документы (картинки) без OCR-слоя не поддерживаются.
- Требуется современный браузер с поддержкой ES6 и FileReader.
